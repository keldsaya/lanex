# LANEX
**Lanex** is a monolithic, multiprocessor, minimalistic console-based operating system kernel for the x86_64 architecture, built from absolute scratch.
The name "Lanex" originated from a friend's reaction to Linux memes — a simple, memorable twist that stuck.

## Features
- **Monolithic Architecture:** Core services run in privileged mode for maximum performance.
- **x86_64 (AMD64):** Targets modern 64-bit processors.
- **Symmetric Multiprocessing (SMP):** Designed to utilize multiple CPU cores.
- **Minimalistic & Console-First:** Focus on core kernel functionality with a console interface — no GUI (yet).
- **Built from Scratch:** No reliance on existing large codebases; every line is hand-crafted for learning and control.

## Philosophy
Lanex is a learning journey and a low-level programming endeavor. It prioritizes simplicity, clarity, and understanding over enterprise features. It's about building a foundation and exploring how operating systems work under the hood.

## Current Status
**Heavy Development / Pre-Alpha** 
The kernel is in its early stages. Core components are being actively implemented and may change frequently. Expect bugs, incomplete features, and ongoing experimentation.
